Bookmarklets

The Chrome Extension. But Better.

Bookmarklets are JavaScript Commands in the form of a hyperlink, or Browser Bookmark. They are similar to extensions, in that they add functionality to the browser, but they are different in that Extensions have access to most, if not all the browser. Whereas Bookmarklets inject code into a website, running as if the programmers of the website put the code into it. One Example of a Bookmarklet is Picture in Picture, which on one of my computers, I have installed a Chrome Extension for, but on my other computer, I just use the Bookmarklet.

Because Bookmarklets run JavaScript from a Website, They are generally less dangerous of being malicious than an Extension, Due to the overall lack of powerful Browser Controlling APIs.
